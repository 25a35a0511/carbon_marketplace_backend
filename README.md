# 🌍 Carbon Marketplace Backend

A scalable backend API for a Carbon Credit Marketplace that enables users to list, buy, and manage carbon credits.

---

## 🚀 Features

* 🔐 User Authentication & Authorization
* 📦 Carbon Credit Management
* 🏗️ Project Listing & Approval
* 🧑‍💼 Seller & Admin Roles
* 📩 Contact/Support API
* 🛡️ Security Middleware (Helmet, Rate Limiting, Sanitization)
* 📊 Logging & Error Handling

---

## 🧱 Project Structure

```
src/
│── config/        # Configuration files (DB, env, etc.)
│── controllers/   # Business logic
│── middleware/    # Custom middleware (auth, error handling)
│── models/        # Database models
│── routes/        # API routes
│── seed/          # Seed data
│── utils/         # Utility functions
│── app.js         # Express app setup
│── server.js      # Entry point
```

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* JWT Authentication
